1 Напишите запрос, который может вывести всех покупателей, чьи имена начинаются с буквы,
попадающей в диапазон от A до G.
  SELECT * FROM cust WHERE REGEXP_LIKE (cname, '^[a-g]');

2 Напишите запрос, который выберет всех покупателей, имена которых содержат букву e.
  SELECT * FROM cust WHERE cname LIKE '%e%';

3 Напишите запрос, который сосчитал бы сумму всех заказов для покупателя 2008
  SELECT SUM(amt) FROM ord Where cnum Like '2008';

4 Напишите запрос, который сосчитал бы число различных значений поля rating в таблице
Покупателей.
  SELECT COUNT(DISTINCT rating) FROM cust;

5 Напишите запрос, который выбрал бы наибольший заказ для покупателей с номерами 2004 и
2006
  SELECT cnum, MAX(amt) FROM ord where cnum = 2004 or cnum = 2006 group by cnum; 

6 Напишите запрос, который выбрал бы продавца, чье имя является первым в алфавитном
порядке среди имен, содержащих букву i.
  SELECT min(cname) FROM cust WHERE cname LIKE '%i%';

7 Напишите запрос, который выбрал бы самый ранний заказ для каждого продавца.
  SELECT min(odate), snum FROM ord group by snum
